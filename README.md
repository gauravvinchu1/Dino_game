# Dino Hunter: Survival

A 2D top-down dinosaur survival game built with vanilla HTML, CSS, and JavaScript.

## Gameplay
- **Objective**: Hunt smaller dinosaurs to grow and avoid being eaten by larger ones.
- **Goal**: Become the apex predator and achieve the highest score possible.

## Controls
- **WASD / Arrow Keys**: Move your dinosaur.
- **Touch**: Use the virtual joystick on mobile devices.

## Features
- **Procedural Graphics**: Dinosaurs are drawn dynamically using HTML5 Canvas.
- **Web Audio API**: Real-time sound generation for eating and extinction events.
- **Growth System**: Your dinosaur grows in size and strength as you hunt.
- **Difficulty Scaling**: Enemies become faster and more frequent over time.
- **High Score**: Your personal best is saved via local storage.

## Development & Deployment

### Local Development
To run the game locally, simply open the `index.html` file in any modern web browser.

### Deploying to Vercel
This project is configured for instant deployment to Vercel as a static site.

#### Using Vercel CLI:
1. Install Vercel CLI: `npm install -g vercel`
2. Run `vercel` in the project root.
3. Follow the prompts to deploy.

#### Using GitHub Integration:
1. Push this project to a GitHub repository.
2. Connect your GitHub account to Vercel.
3. Select this repository and click **Deploy**.

No build step is required.
